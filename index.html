Text: Font and Shadow

Font
For custom fonts, install the fonts using google and use the api link in the head of your document
Text Shadows
Use the text-shadow property in CSS to give your text a shadow. Remember they can stack! The first 2 numbers determine the distance and angle of the shadow, and you can add a color.
Bootstrap
Use 'npm install bootstrap' and link bootstrap to your document and start type div 50,000 times a day, today!
